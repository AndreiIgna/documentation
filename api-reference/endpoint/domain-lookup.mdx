---
title: "Domain Lookup"
description: "Check domain availability and other infos"
api: 'GET /domains/{domain}'
---

This endpoint builds on top of the RDAP / WHOIS data. It normalizes fields between TLDs, formats data to the same format and returns the most accurate data after checking both Registry and Registrar RDAP/WHOIS.

Use this endpoint to get availability, registrant, status and more detailed info.
